Office Account

How to create an office account.

Quickly learn how create an office account through these simple steps
1 Go to Administrator panel then click on the office accounts. option pill, Then Choose create Office account, from the options provided in order to load the Registration form.
2 Select or create a Service unit Account i.e. Ministry, Organisation, or any other business firm under which you want your office account to be created.
3 Select or create a Branch Account under which you want your office account to be created.
4 Select or create a Department Account under which you want your office account to be created.
5 Now finally create Office Account by filling in the form provided to you.

User Account

How to create or add user to the system.

Quickly learn how create a user account, or add a user to the system through these simple steps
1 Go to Administrator panel then click on the User accounts. option pill.
2 Select Add user from the options provided to load the User creation / or staff registration form.
3 Select Office Account under which you want to register the user.
Note: If you are not yet aware of how to create an office account, then follow the steps in the section above this on How to create an Office Account
4 Now finally add user or create User Account by filling in the form provided to you.
